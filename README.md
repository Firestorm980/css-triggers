# CSS Triggers

Tells you what gets triggered by mutating a given CSS property.

## Running application locally

1. Clone the project.
1. Run `npm install` in __css-triggers__.
1. Run `npm run start`
1. Go to [localhost:3000](http://localhost:3000/)
